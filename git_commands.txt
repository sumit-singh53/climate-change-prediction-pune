# Git Commands to Push Your Climate Dashboard

# 1. Initialize Git repository (if not already done)
git init

# 2. Add all files
git add .

# 3. Commit the changes
git commit -m "Pune Climate Dashboard - Complete project with 151K+ records"

# 4. Create GitHub repository
# Go to https://github.com and create a new repository named: pune-climate-dashboard

# 5. Add remote origin (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/pune-climate-dashboard.git

# 6. Push to GitHub
git branch -M main
git push -u origin main

# 7. Deploy to Streamlit Cloud
# - Go to https://share.streamlit.io
# - Sign in with GitHub
# - Click "New app"
# - Select your pune-climate-dashboard repository
# - Set main file path: streamlit_app.py
# - Click "Deploy!"

# Your public URL will be:
# https://YOUR_USERNAME-pune-climate-dashboard-streamlit-app-RANDOM.streamlit.app
